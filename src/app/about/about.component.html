About Formulator

This application was developed using the following technologies:

  • Angular Version: 15.2.10
  • Node.js Version: 18.12.0
  • MySQL Workbench 8.0 CE
  • NVM (Node Version Manager) 1.1.12
  • JDK (Java Development Kit) 17

- Angular is a frontend framework used for building single-page applications
 (SPAs) with TypeScript.

- Node.js is a JavaScript runtime used for server-side development, while
 MySQL Workbench is a tool for managing MySQL databases.

- NVM (Node Version Manager) allows developers to manage multiple
 versions of Node.js on a single machine.

- JDK (Java Development Kit) 17 is a software development kit used for
 developing Java applications.
